## Fully Implemented Features
- CPU
- SSU
- RTC
- [SCI3 IR Communication](https://github.com/Ceravia/melonDS-IR)
- Peripherals
  - LCD
  - Eeprom
  - 3-Button Input
  - Beeper Audio

## Partially Implemented Features
- ADC
- Accelerometer

## Planned Features
- Sleep Mode
- Walk Actions (Accelerometer)
- Peer Play
- Gifts
